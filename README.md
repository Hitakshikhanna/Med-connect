Med-Connect 🏥
Med-Connect is a full-stack web application designed to help users track their health parameters, monitor medical logs, and calculate health metrics efficiently.

🚀 Features
Medicine Tracker: Keep log updates of daily medicine schedules.

BMI Calculator: Calculate body mass index effortlessly with instant feedback.

Contact & Support: A dedicated page for user inquiries and medical connectivity.

🛠️ Tech Stack
Frontend: HTML5, CSS3, Bootstrap (Responsive Design)

Backend: Node.js, JavaScript (Express server architecture)

📁 Project Structure
index.html - Home landing page

bmi.html - Health metric calculations

medicine.html - Tracker dashboard

test.js - Backend server execution environment
